# 앱 이름

```
이게뭐약-AI기반 알약 정보 검색 및 관리
```

# 간단한 설명

```
보건/의료 관련 업계 종사자와 약 복용자를 위한 AI 기반 알약 정보 검색 및 관리 애플리케이션
```

# 자세한 설명

```
[이미지 검색]
AI 모델 (구글 Gemini)로 사용자가 촬영한 알약의 사진에서 외형 정보를 추출하여 알약의 정보를 검색

[식별 검색 (개선중)]
알약의 이름, 제조사, 외형 정보(식별 정보)를 통해 알약의 정보를 검색

[통합 검색 (준비 중)]
제품명이나 성분명, 증상 등으로 쉽게 알약의 정보를 검색

[주변 약국 (준비 중)]
지금 찾아갈 수 있는 주변의 약국에 대해 안내

[알약 보관함]
검색한 알약의 정보를 다음에 쉽게 찾을 수 있도록 저장

[알약 카운팅 (준비 중)]
AI 모델을 활용하여 사용자가 촬영한 이미지 내 알약의 개수를 카운팅


※ 본 애플리케이션은 정부 기관(식품의약품안전처 등)과 공식적인 연관이 없는 민간 서비스입니다. 제공되는 정보는 의약품안전나라, 공공데이터포털, 식의약 데이터 포털의 데이터를 기반으로 하며, 최신 정보는 해당 기관의 공식 웹 사이트에서 해주시기 바랍니다.
- 의약품안전나라 (https://nedrug.mfds.go.kr/index)
- 공공데이터포털 (https://www.data.go.kr/)
- 식의약 데이터 포털 (https://data.mfds.go.kr/)


※ 본 애플리케이션이 활용하는 정보
- 의약품안전나라 / 의약품 낱알 식별 (https://nedrug.mfds.go.kr/pbp/CCBGA01/getItem?totalPages=8&limit=10&page=2&&openDataInfoSeq=11)

- 의약품안전나라 / 의약품 허가 상세 정보 (https://nedrug.mfds.go.kr/pbp/CCBGA01/getItem?totalPages=8&limit=10&page=2&&openDataInfoSeq=12)

- 공공데이터포털 / 식품의약품안전처_의약품 제품 허가 정보 (https://www.data.go.kr/data/15095677/openapi.do#/API%20%EB%AA%A9%EB%A1%9D/getDrugPrdtPrmsnDtlInq04)

※ 개인정보 처리방침 (https://sites.google.com/view/wip-personal-info-policy?usp=sharing)
```
