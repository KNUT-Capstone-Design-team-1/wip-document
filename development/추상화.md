#### 알약 식별 검색 (pill_identification_search)

- 알약의 외형적인 특징을 입력하여 조건이 포함되는 알약을 검색

#### 알약 이미지 검색 (pill_image_search)

- 알약의 이미지를 첨부하여 조건이 포함되는 알약을 검색

#### 알약 검색 결과 목록 (pill_search_result_list)

- 알약 식별 검색 및 알약 이미지 검색 결과 목록

#### 알약 상세 정보 (pill_search_result_detail)

- 알약의 상세 정보

#### 주변 약국 (nearby_pharmacy)

- 사용자의 위치 반경 내 약국들을 지도에 표시

#### 알약 보관함 (pill_save)

- 검색된 알약을 사용자의 기기에 저장하여 검색 수행 없이 정보를 확인
